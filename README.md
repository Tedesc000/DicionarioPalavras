# VLAD 2.0 — Virtual Language Assistant and Dictionary

O **VLAD 2.0 (Virtual Language Assistant and Dictionary)** é um projeto que busca facilitar a interação de usuários com textos e dicionários online.
O projeto integra um **contador de elementos textuais** e um **dicionário interativo**, permitindo que o usuário consulte palavras diretamente a partir de um texto digitado.

---

## Objetivo do Projeto

O principal objetivo do VLAD 2.0 é **facilitar a leitura e a escrita**, especialmente em contextos educacionais, oferecendo:

- Contagem automática de palavras, números e frases
- Seleção interativa de palavras dentro do texto
- Consulta instantânea a definições e significados
- Estudo prático de **DOM**, **RegEx**, **APIs** e **requisições HTTP**
---

## Contexto e História

A ideia do VLAD surgiu a partir de uma apresentação realizada para turmas do **Ensino Médio** sobre o projeto **RevisãoOnline**, uma plataforma gratuita voltada para revisão e correção de redações no modelo ENEM.

Durante essas apresentações, foi observado que muitos alunos apresentavam dificuldades relacionadas a:
- Palavras desconhecidas
- Escrita incorreta

Inicialmente, o VLAD conseguiu integrar ao RevisãoOnline o dicionário online **Dicio**. No entanto, a criação do VLAD 2.0 surgiu da necessidade de uma solução mais integrada, gratuita e flexível.

O **VLAD 2.0** representa uma evolução dessa ideia inicial, com foco maior em integração, interatividade e estudo técnico.

## Tecnologias Utilizadas

- **Wikimedia API (Wiktionary)** — Fonte de dados do dicionário
- **DOM API** — Manipulação dinâmica da página
- **RegEx** — Processamento e análise textual

---

## Aprendizados Envolvidos

O desenvolvimento do VLAD 2.0 envolveu o estudo e aplicação prática de:

- Manipulação do DOM
- Requisições HTTP e consumo de APIs públicas
- Tratamento de dados retornados em formato JSON
- Conversão de strings HTML em estruturas DOM
- Expressões regulares para análise textual
- Organização de interface e experiência do usuário (UX)
